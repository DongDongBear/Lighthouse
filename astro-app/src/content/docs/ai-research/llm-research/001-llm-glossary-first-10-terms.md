# 001. 小白词典：10 个最常见术语（一次搞懂）

你不需要一次记住所有术语。先记这 10 个，已经能读大部分入门文章。

---

## 1) Prompt（提示词）

你给模型的输入文本。

你问得越清楚，模型通常答得越好。

## 2) Token

模型处理文字的最小单位。

可以粗略理解成“字/词片段”。

## 3) Context Window（上下文窗口）

模型一次能“看到”的最大 token 数。

超出就会忘前面内容。

## 4) Hallucination（幻觉）

模型编造了看似合理但错误的信息。

## 5) Parameters（参数）

模型内部的“记忆权重”。参数越多，潜在能力通常越强，但不是绝对。

## 6) Pretraining（预训练）

在海量文本上学习语言能力的阶段。

## 7) Fine-tuning（微调）

在特定数据上继续训练，让模型更符合某个任务。

## 8) RL（强化学习）

通过“尝试-打分-调整”让模型行为变好的训练方式。

## 9) RAG

检索增强生成。先查资料，再回答。

## 10) Agent

会“自己分步骤做事”的 LLM 系统，比如会调用工具、反复搜索。

---

## 一句话串起来

你写 Prompt，模型在 Context Window 内按 Token 生成答案。它基于 Parameters，经过 Pretraining 和 Fine-tuning，可能还经 RL 强化。为了减少 Hallucination，可以用 RAG；再进一步就是 Agent。
